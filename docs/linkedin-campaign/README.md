# LinkedIn Campaign Reference

This folder contains the LinkedIn campaign package for the AppSec vulnerability management lab.

## Files

- `final-posting-guide.md`: publish-ready campaign guide with cadence, hooks, first comments, engagement prompts, and checklist.
- `linkedin-drip-campaign.md`: full three-post campaign draft with technical evidence notes.
- `campaign-summary.md`: strategic summary, audience framing, voice guidance, and technical accuracy checklist.
- `thumbnails/`: final PNG and SVG thumbnail assets for each post.

## Campaign Arc

1. **From Deployment to a Real AppSec Program**
   - Thumbnail: `FIVE ISSUES. ONE APP.`
2. **A Finding Is Not Yet Remediation Work**
   - Thumbnail: `A FINDING IS NOT A FIX`
3. **Why "Fixed" Still Needs Proof**
   - Thumbnail: `FIXED IS NOT ENOUGH`

The campaign is intentionally story-first. The goal is to show how vulnerability management discipline transfers into AppSec: identify, validate, prioritize, assign, remediate, re-test, and prove closure.
